# CLI Reference

Interactive menu-driven command-line interface for managing Artic trading agents.

<Callout type="info" title="Session-based">
The CLI saves your JWT token to <code>~/.artic/token</code>. Subsequent launches skip login until the token expires.
</Callout>

## Setup

```bash
cd clients/cli
pip install -r requirements.txt
python -m cli
```

## Authentication

On first launch:

<DocTable
  headers={["Option", "Action"]}
  rows={[
    [<strong>1 — Login</strong>, "Authenticate with email + password"],
    [<strong>2 — Register</strong>, "Create account and auto-login"],
    [<strong>0 — Exit</strong>, "Quit"],
  ]}
/>

Tokens persist in `~/.artic/token`. Option 0 clears the saved token.
